# Vasave Cafe Website

A premium, static cafe website built with pure HTML, CSS, and JavaScript.

## Features
- **Responsive Design**: Works on mobile, tablet, and desktop.
- **Dynamic Menu**: Product grid populated via JavaScript.
- **Shopping Cart**: Fully functional cart using `localStorage`.
- **Product Details**: Detailed view for every item.
- **Premium Aesthetic**: Custom CSS design system.

## Setup Instructions
1. This is a static website. No server or backend is required.
2. Simply double-click `index.html` to open it in your browser.
3. Ensure you have an internet connection to load the images (Unsplash).

## Folder Structure
- `css/style.css`: All styles.
- `js/script.js`: Logic for cart, menu, and interactions.
- `*.html`: Site pages.
